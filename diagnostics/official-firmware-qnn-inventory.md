# nubia Z80 Ultra official firmware QNN inventory

調査日: 2026-07-13

## 結果

ローカルのワークスペースおよびユーザーのDownloadsには、NX741J / Z80 Ultraの公式フルOTAまたはファームウェアパッケージは存在しなかった。

nubia公式サポートページ、公式マニュアルページ、およびnubia公式ROM配布ドメインを確認したが、NX741J / Z80 Ultraの合法的に直接取得可能なフルOTA/ファームウェアパッケージは特定できなかった。非公式ROMサイト、OTA URL取得の回避手段、認証回避は使用していない。

したがって、オフラインvendor image展開は実施しておらず、ファームウェア内のQNN version、build ID、interface version、build dateは未取得である。端末への書き戻しも行っていない。
